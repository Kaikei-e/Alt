# ADR-000691: feeds/search と /search を Alt-Paper デザイン言語で刷新する

## Status

Accepted

## Date

2026-04-11

## Affected Services

- alt-frontend-sv (TypeScript / SvelteKit)

## Context

Alt-Paper デザイン言語への段階的 UI 刷新を進めてきた。これまでに Ask Augur ([[000674]])、Tag Trail ([[000678]])、Morning Letter ([[000680]])、Dashboard ([[000682]])、Feeds ([[000686]])、Mobile FeedDetails ([[000687]]) の 6 画面をリデザイン済みだった。

残る検索系 2 画面（`/feeds/search` と `/search`）は旧スタイルが残存していた。

- **glassmorphism**: `glass` クラス、`--alt-glass`、`backdrop-blur`、`rounded-[24px]`
- **Lucide icons**: Search, Loader2, Loader, SquareArrowOutUpRight, AlertTriangle, FileText, BookOpen, Tag, ChevronRight の 9 種
- **shadcn コンポーネント**: `Button`, `Input`（`$lib/components/ui/` から import）
- **装飾的アニメーション**: `hover:-translate-y-[2px]`, `hover:scale-[1.02]`, `active:scale-[0.98]`
- **ハードコード色**: `#ffffff`, `#d0d0d0`, `#dc2626`, `amber-*`, `red-*`
- **絵文字**: 💡, 🔍, ✨
- **Tailwind ユーティリティでの editorial styling**: `text-[var(--text-secondary)]` 等のインライン
- **グラデーション skeleton**: 90deg linear-gradient ベースの skeleton-pulse

これらは Alt-Paper 準拠の他画面と視覚的に断絶しており、統一感を損なっていた。

## Decision

Responsibility-Driven Design に基づき、各検索画面の責務に最適な Alt-Paper 表現を適用する。

### 責務定義

| Route | 責務 | メタファー |
|-------|------|-----------|
| `/feeds/search` | RSS 記事アーカイブ検索 | **Archive Desk** — 新聞社の切り抜き資料室 |
| `/search` | 横断的ナレッジ検索（記事・要約・タグ） | **Reference Desk** — 編集部のリサーチデスク |

### `/feeds/search` "Archive Desk"

- `PageHeader` を editorial header に置換（monospace date + serif "Archive Desk" + 1px rule）
- shadcn `Input` / `Button` を native `<input>` / `<button>` + scoped CSS に置換
- Lucide `Search`, `Loader2` を削除、pulsing dot (8px, 1.2s) + italic text に置換
- モバイル 4 コンポーネント (SearchFeedsClient, SearchWindow, SearchResults, SearchResultItem) を全面書き換え
  - glassmorphism → flat 1px border container
  - `rounded-2xl` / `rounded-full` → sharp edges (border-radius: 0)
  - `SquareArrowOutUpRight` icon → title をリンク化
  - 絵文字 → italic text
  - 装飾的 hover → `transition: background 0.15s` のみ
- stagger entry animation: desktop 60ms, mobile 40ms
- page reveal animation: 0.4s fade + translateY(6px)
- `data-role` attributes を全コンポーネントに付与
- 既に Alt-Paper 準拠の `DesktopFeedCard` と `FeedDetailModal` はそのまま再利用

### `/search` "Reference Desk"

- `PageHeader` を editorial header に置換（monospace date + serif "Reference Desk" + 1px rule）
- Lucide icons 6 種を全削除（Search, AlertTriangle, FileText, BookOpen, Tag, ChevronRight）
- `ChevronRight` → `>` text character
- `rounded-lg` / `rounded-full` → sharp edges
- amber/red ハードコード警告バナー → terracotta left stripe (`.ref-degraded-banner`)
- gradient skeleton pulse → opacity-based (`.skeleton-bar`, 1.2s)
- section label を monospace uppercase に統一（`ARTICLES`, `RECAPS`, `TAGS`）
- field badge を flat border（rounded なし）、tag token を flat background に変更
- stagger entry animation を全 section hit に適用

### 共通適用パターン

- **Loading**: `.loading-pulse` (8px dot, `pulse 1.2s ease-in-out infinite`)
- **Error**: `.error-stripe` (3px left border `var(--alt-terracotta)`)
- **Button**: `1.5px solid var(--alt-charcoal)`, uppercase, 44px min-height
- **Input**: `1px solid var(--surface-border)`, focus で charcoal、`font-size: 1rem`（iOS zoom 防止）
- **Scoped CSS**: editorial styling は全て `<style>` ブロック内、Tailwind はレイアウト (flex, grid, gap) のみ
- **Reduced motion**: `@media (prefers-reduced-motion: reduce)` で全アニメーション無効化

## Consequences

### Pros

- 検索系 2 画面が Alt-Paper デザイン言語に準拠し、他の 6 画面と視覚的に統一された
- shadcn `Button` / `Input` 依存を除去し、bits-ui CSS コンパイルエラーの影響範囲が縮小した
- Lucide icons 9 種の import を除去し、バンドルサイズが微減した
- 31 の新規テストが Alt-Paper 準拠を検証可能にした
- `data-role` attributes により、CSS クラス変更に影響されない安定的なテストセレクタが確保された
- iOS Safari 対応（`font-size: 1rem`, `100dvh`, safe-area padding）が一貫して適用された

### Cons / Tradeoffs

- `.loading-pulse`, `.error-stripe`, `@keyframes` 等の共通パターンが各コンポーネントの scoped style に複製されている。Alt-Paper の慣習（各コンポーネント self-contained）に従った意図的な選択だが、将来的にパターン変更時は複数ファイルの更新が必要
- `FloatingMenu` は Lucide icons を含んだままであり、別タスクでの Alt-Paper 化が必要
- `RecapPreviewModal` の Alt-Paper 準拠確認は未実施（今回のスコープ外）
- モバイル `SearchResultItem` の summary 展開 UI は機能的に維持したが、FeedDetails のような streaming summarization には対応していない（既存の制約を踏襲）
